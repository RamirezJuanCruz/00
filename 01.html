3-¿Qué es una etiqueta? ¿Funciona si se escribe tanto como mayúscula como en minúscula? ¿Cuál es la major práctica para escribir etiquetas?

Una etiqueta es un componente del documento que contiene el contenido, estando la información entre una etiqueta de apertura y una etiqueta de cierre. La etiquetaes capaz de funcionar escrita de ambas modos, sin embargo, por temas de comodidad las etiquetas siempre se escriben en minúscula. La manera más practica de escribir una etiquera es hacerlo en orden. Primero se pondría el título del más importante al menos importante (h1-h6), después se pondrían los parrafos (p) debajo de sus respectivos titulos y por ultimo se cerrarían todas las etiquetas usadas

4-¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes

A lo que llamamos elemento HTML es una serie que se utiliza para encerrar, delimitar o marcar diferentes partes del contenido para hacer que aparezcan de una cierta manera, o actúen de determinada forma. Su anatomía esta formada de la siguiente manera. Etiqueta de apertura, el contenido y la etiqueta de cierre.

11- ¿Para qué sirve la etiqueta img? ¿Cuáles son sus atributos? ¿Para qué sirven cada uno de ellos?

La etiqueta img se utiliza para representar una imagen en el documento. Sus atributos son: "align" sirve para el alinamiento del texto respecto al contexto que la rodea. "alt" sirve para definir el texto alternativo que describe la imagen, texto que los usuarios verán si la URL de la imagen es errónea o la imagen tiene un formato no soportado o si la imagen aún no se ha descargado. "border" sirve para Anchura del borde alrededor de la imagen. "crossorigin" se encarga de indicar si la búsqueda de la imagen debe ser por CORS o no. "anonymous" Se encarga de pedir una petición cross-origin. "use-credentials" sirve por sí el servidor no emite credenciales al sitio de origen, la imagen será pintada y su uso restringido. "height" se encarga de la altura de la imagen. "hspace" se encarga del número de píxeles de espaciado a la izquierda y la derecha de la imagen. "ismap" se encarga de indicar que la imagen es parte del mapa del lado del servidor. Así que, se envían las coordenadas precisas de un clic. "longdesc" usa la URL como descripción de una imagen mostrada, complementa al texto de "alt". "name" es el nombre para el elemento. Soportado en HTML4 solo para compatibilidad con versiones anteriores. "referrerpolicy" Una cadena indicando que referencia usar cuando buscas un recurso, entre ellos esta "no-referrer", "no-referrer-when-downgrade", "origin", "origin-when-cross-origin" y "unsafe-url". "sizes" es a lista de una o más cadenas separadas por comas indicando el tamaño de la fuente. Cada tamaño de la fuente consiste en: Codición de medios. Debe omitirse en el último ítem y valor del tamaño. "width" se usa para generar el ancho de la imagen en píxeles CSS en HTML5, o píxeles o porcentaje en HTML4. "usemap" se usa para para URL parcial (empezando con '#') de un mapa de imagea (en-US) asociado a un elemento."vspace" se usa para generar el número de píxeles de espacio blanco insertado sobre y bajo la imagen.
