1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

ul sirve para crear una lista desordenada, siendo un ejemplo una lista en donde se ponga las tareas del día:

  • Limpiar la cocina
  • Estudiar
  • Pasear al perro

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

ol sirve para definir listas o viñetas ordenadas, bien con numeración o alfabéticamente. Un ejemplo en donde se podría usar sería el orden en el cuál ver una saga de peliculas:

  1. 1er Ejemplo
  2. 2do Ejemplo
  3. 3er Ejemplo

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

Esta etiqueta sirve para representar una lista descriptiva. Un ejemplo de donde usarlo sería para escribir una definición:

Firefox
A free, open source, cross-platform, graphical web browser developed by the Mozilla Corporation and hundreds of volunteers.

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

Esta etiqueta sirve para declara cada uno de los elementos de una lista. Un ejemplo de donde usarlo usarlo sería para una lista de las compras:

  • Pasta de dientes
  • Shampoo
  • Jabón

2.1- ¿qué es una url absoluta?

Una URL absoluta es aquella que contiene toda la información necesaria para localizar un recurso

2.2- ¿qué es una url relativa?

Una URL relativa es la que se encarga de localizar un recurso mediante una dirección URL absoluta como punto de partida
